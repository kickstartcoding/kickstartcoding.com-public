Static Sites & Bootstrap

Automation: Bash, Git, and Python Scripts

Learn how to use Git to keep your code safe under version control. Learn effective terminal use, and how to write Bash and Python scripts to replace tedious tasks.

Fundamentals

Computer Programming with Python

Begin a classic introduction to backend programming with Python, learning core data types and control-flow structures. Go deeper with OOP, Pipenv, and Jinja templating.

Backend

Intro to Networking with Python

Learn about the Internet, networking, and server-side programming. You will use web APIs, implement HTTP over TCP/IP, and finally basic request routing and web apps.

Backend

Practical Django

Learn the best practices from a 10+ year Django industry veteran: MVC, CRUD, forms, urls, users, admin, models, data cardinality (eg Many-To-Many), unit testing, and much more.

Backend

Intro to Databases with Postgres SQL

Learn SQL basics on a Postgres DB, before diving into JOINs, constraints, query analysis, indexes and optimization, trigram-based full-text search, geodata, and more.

Frontend

Modern Web: Vanilla JS

Learn modern JavaScript, including let, const, iterators, and DOM manipulation. Dive into tricky CSS concepts like ancestor positioning and pseudo-elements.

Frontend

Thinking in React

In this hooks-based React course, you go beyond just learning the JSX syntax, and practice "thinking in React" with data-flow, lifting state, and more.

Frontend

Full Stack JS: MERN, Redux, and Routing

Take your React JS skills to the full stack by learning MongoDB, Express.JS, and Node.js. Learn Redux for state management, and Routing for multi-page apps.

Career

Algorithms: Computer Science for Job Seekers

A CS course hyper-focused on whiteboard interviews. Learn Big O, time complexity analysis, sorting, dynamic programming, recursion, coding puzzle techniques, and more.

Career

Career Capstone

Use a 5-milestone process to build a software product from specs to launch. Learn how tech roles interlock, including agile Product Management and DevOps.
