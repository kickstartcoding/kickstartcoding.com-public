Learning Center

Free resources for learning to code

Welcome to the Kickstart Coding Learning Center. Here you'll find cheatsheets, articles, tutorials, events, and other resources to help you in your journey to learn to code. We want you to have access to the highest quality resources, regardless of if you are learning with us or by yourself!

Events

Community is one of the building blocks of success at Kickstart Coding. We regularly host informal study sessions and free workshops that students and community members alike can attend. Regardless of if you're a prospective student, or just a life-long learner, all are welcome in our community!

Free Coding Lessons

Kickstart Coding created FreeCodingLessons.com as a free resource for both our students and the wider code learning community.

  • Learn to build a website in only 3 FREE self-guided lessons
  • Join an Online or Oakland-based study group to get help
  • It's perpetually a work in progress, so be sure to keep on checking back as we add more and more coursework, resources, and useful material!
